***Переменная*** - именованный участок памяти, который может хранить значение, которое может быть изменено программой

```c
// О.ф.
// тип имя;

int a;
double x, y, z;
```

Инициализация - присвоение начального значения

```c
// О.ф.
// тип имя = значение;
int a = 1;
double x, y = 0.1, z;
float q = 1.23f;
```
